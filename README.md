# 🗂️ A Sua ID — CRM

Sistema web desenvolvido internamente para centralizar e monitorar todas as interações com clientes da **A Sua Identidade Digital**.

> Projeto desenvolvido com assistência de IA (Vibe Coding) e evoluído continuamente com foco em resolver problemas reais do negócio.

---

## 🚀 Funcionalidades

| Módulo | Descrição |
|---|---|
| 📊 **Dashboard** | Visão geral com métricas e indicadores de performance |
| 👥 **Clientes** | Cadastro completo e gestão de clientes |
| 💬 **Interações** | Histórico de todas as interações com clientes |
| 📋 **Pipeline de Vendas** | Acompanhamento visual do funil de vendas |
| 📅 **Agendamentos** | Formulário e controle de agendamentos |
| ✅ **Tarefas** | Gerenciamento de tarefas por agente |
| 📝 **Notas** | Anotações vinculadas a clientes |
| 📣 **Campanhas** | Criação e acompanhamento de campanhas |
| 🔄 **Renovações** | Controle de renovações de certificados |
| 📈 **Relatórios** | Relatórios detalhados de desempenho |
| 👤 **Performance** | Análise de desempenho por agente |
| 💬 **Chat** | Módulo de comunicação interna |
| 🔗 **Google Sheets** | Sincronização com planilhas Google |
| 📥 **Importação** | Importação de dados em massa |
| 🧹 **Limpeza de Base** | Identificação e remoção de duplicatas |
| ⚙️ **Admin** | Painel administrativo de controle |

---

## 🛠️ Tecnologias utilizadas

- **[React 18](https://react.dev/)** — Interface de usuário
- **[Vite](https://vitejs.dev/)** — Build tool e servidor de desenvolvimento
- **[Tailwind CSS](https://tailwindcss.com/)** — Estilização
- **[Radix UI](https://www.radix-ui.com/)** — Componentes acessíveis
- **[React Router v6](https://reactrouter.com/)** — Navegação
- **[TanStack Query](https://tanstack.com/query)** — Gerenciamento de estado e cache
- **[React Hook Form](https://react-hook-form.com/)** + **[Zod](https://zod.dev/)** — Formulários e validação
- **[Recharts](https://recharts.org/)** — Gráficos e visualizações
- **[Framer Motion](https://www.framer.com/motion/)** — Animações
- **[Base44 SDK](https://base44.com/)** — Backend e autenticação
- **[jsPDF](https://github.com/parallax/jsPDF)** + **[html2canvas](https://html2canvas.hertzen.com/)** — Exportação de relatórios

## ⚙️ Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/Feedelfino/asuaid-crm.git

# Entre na pasta
cd asuaid-crm

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no navegador.
